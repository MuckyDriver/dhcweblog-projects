NukeLab 2018

Nuke Born Watch Event by Claasgreeneye Watch the bombing nuke explode island "Craiz" with buildings which will fall into pieces.

About Game

NukeLab was the first game I created on Roblox that I fully finished. I had started many more projects before this one but this was the first one to be completed. I advertised this game quite a lot and that's probably the reason for the high number of visits it has. All you do in this game is nuke an island by pressing a big red button - you watch the island getting destroyed. You are capable of redoing this as many times you want via respawning the nuke and the island.
