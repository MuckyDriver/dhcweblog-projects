Planetary Object Realm 2

Explore the planetary objects of our solar system and beyond. Planetary Object Realm 2 provides a revised version of the original with more content and features.

About Game

I started building this game because there was another version of it which I had made a long time ago however I thought I could recreate it with better features and user experience. The main part of the game was fully complete within a week of development. The purpose of this experience is to learn about our solar system. The whole idea around planetary object realm started when I was doing a college project revolving around making an application to showcase information about the solar system.
