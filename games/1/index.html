Mucky Driver

Mucky Driver is a game where you upgrade through various cars and vehicles by collecting Points, Gems and CustomPoints all over the map.

About Game

The long lasting project Mucky Driver is my all time biggest project I did on Roblox. The development of the game started when I wanted to recreate a game called Offroad Sim 19. The main features of the game were complete however I could never get it polished enough to advertise. Bugs and lack of motivation kept getting in my way so eventually the project was put to rest. If you do play this game please be aware that there are many bugs you will encounter.
